# 📚 Documentación — Bootcamp Dart

Índice de la documentación transversal del bootcamp (no específica de una semana).

- [🗓️ Plan de Estudios](plan-estudios.md) — temario semana a semana, fases y objetivos
- [📦 Versiones del Stack](stack-versions.md) — versiones pineadas del SDK de Dart y paquetes
- [⚙️ Configuración del Entorno](setup/README.md) — SDK local y/o Docker

## Para instructores

- Mantener `docs/plan-estudios.md` sincronizado con los `README.md` de cada semana — es la
  fuente de verdad del temario cuando se planifica una cohorte.
- Actualizar `docs/stack-versions.md` al iniciar cada cohorte nueva (fijar versiones exactas,
  no reabrir el rango hasta la siguiente cohorte).
- Dominios anticopia: NO se definen aquí — se gestionan vía `sena/eval-semanal` sobre el
  catálogo compartido `bc-sql/scripts/assign_domains.py` (150 dominios, 15 reservados para
  ejemplos). `Biblioteca` es el dominio canónico usado en toda la teoría/ejemplos de este repo.
