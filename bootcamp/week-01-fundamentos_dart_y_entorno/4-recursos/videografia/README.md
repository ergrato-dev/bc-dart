# Videografía — Semana 01: Fundamentos de Dart

- [Dart programming language — Google I/O talks](https://www.youtube.com/@dart_dev) — canal
  oficial del equipo de Dart en YouTube, incluye charlas sobre null safety y el lenguaje
- [DartPad](https://dartpad.dev/) — no es un video, pero es el sandbox oficial para probar
  snippets de esta semana sin instalar nada, útil mientras ves cualquier tutorial

> 📌 Prioriza fuentes oficiales (canal `dart_dev`) sobre tutoriales de terceros — el lenguaje
> tuvo cambios importantes (sound null safety en Dart 2.12, patterns/records en Dart 3) y
> contenido antiguo puede enseñar sintaxis obsoleta.
