# Videografía — Semana 07: Manejo de Errores y Excepciones

- [Dart programming language — canal oficial](https://www.youtube.com/@dart_dev) — charlas del
  equipo de Dart sobre manejo de errores y patrones de diseño del lenguaje
- [DartPad](https://dartpad.dev/) — no es un video, pero es el sandbox oficial para probar
  `try`/`catch`, excepciones propias y el patrón `Result` sin instalar nada

> 📌 Prioriza fuentes oficiales sobre tutoriales de terceros que mezclan `Result`/`Either` con
> paquetes externos (ej. `dartz`, `fpdart`) — este bootcamp enseña la versión "desde cero" con
> `sealed class` para que entiendas el mecanismo antes de adoptar un paquete que lo empaquete.
