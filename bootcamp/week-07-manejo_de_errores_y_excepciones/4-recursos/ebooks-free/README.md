# Ebooks Gratuitos — Semana 07: Manejo de Errores y Excepciones

- [dart.dev — Error handling](https://dart.dev/language/error-handling) — referencia oficial de
  `try`/`catch`/`finally`, `throw`, `rethrow` y `assert`
- [dart.dev API — Exception class](https://api.dart.dev/stable/dart-core/Exception-class.html) —
  la interfaz marcador que implementan las excepciones personalizadas
- [dart.dev API — Error class](https://api.dart.dev/stable/dart-core/Error-class.html) —
  diferencia entre `Exception` (esperable) y `Error` (bug de programador)
- [dart.dev — Class modifiers: sealed](https://dart.dev/language/class-modifiers#sealed) — base
  para el patrón `Result` visto esta semana

> 📌 Este bootcamp prioriza documentación oficial gratuita sobre libros de pago — el patrón
> `Result` no es sintaxis nueva de Dart sino una convención construida sobre `sealed class` y
> `switch` (semana 6), así que no hay una página única de dart.dev que lo cubra como feature
> aislada.
