# Ebooks Gratuitos — Semana 06: Genéricos y Pattern Matching

- [dart.dev — Generics](https://dart.dev/language/generics) — clases y métodos genéricos,
  bounded generics
- [dart.dev — Records](https://dart.dev/language/records) — referencia oficial de records
  posicionales y con campos nombrados
- [dart.dev — Patterns](https://dart.dev/language/patterns) — catálogo completo de patrones
  (lista, mapa, record, objeto) y destructuring
- [dart.dev — Class modifiers: sealed](https://dart.dev/language/class-modifiers#sealed) —
  sealed classes y exhaustividad

> 📌 Este bootcamp prioriza documentación oficial gratuita sobre libros de pago — genéricos,
> records y pattern matching son features relativamente recientes (records y patterns son de
> Dart 3, 2023), así que la fuente más actualizada es siempre dart.dev.
