# Videografía — Semana 03: Colecciones e Iterables

- [Dart programming language — canal oficial](https://www.youtube.com/@dart_dev) — charlas del
  equipo de Dart, incluye contenido sobre `Iterable` y programación funcional en Dart
- [DartPad](https://dartpad.dev/) — no es un video, pero es el sandbox oficial para probar
  `List`/`Set`/`Map` y cadenas de `where`/`map`/`fold` sin instalar nada

> 📌 Prioriza fuentes oficiales (canal `dart_dev`) sobre tutoriales de terceros — los operadores
> de colección (`if`/`for`/spread dentro de literales) son relativamente recientes en el
> lenguaje, y contenido antiguo puede no cubrirlos.
