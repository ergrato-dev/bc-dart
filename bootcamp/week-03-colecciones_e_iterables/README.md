# Semana 03 — Colecciones e Iterables

> **Fase 2 — POO y Tipos Avanzados** | Semana 3 de 10 | ⏱️ 6 horas

Primera semana de la Fase 2: cómo agrupar y procesar datos con `List`, `Set` y `Map`, y el
estilo funcional (`where`/`map`/`fold`) que reemplaza bucles imperativos verbosos — la base para
modelar colecciones de objetos cuando lleguen las clases en la semana 4.

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- [ ] Crear, mutar y recorrer `List` y `Set`, y explicar cuándo usar cada uno
- [ ] Crear un `Map`, acceder por clave con seguridad y recorrer `keys`/`values`/`entries`
- [ ] Construir colecciones con collection-if, collection-for y spread (`...`, `...?`)
- [ ] Encadenar `where`, `map`, `fold` y `reduce` para procesar colecciones sin bucles manuales
