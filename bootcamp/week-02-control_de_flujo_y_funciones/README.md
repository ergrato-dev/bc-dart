# Semana 02 — Control de Flujo y Funciones

> **Fase 1 — Fundamentos del Lenguaje** | Semana 2 de 10 | ⏱️ 6 horas

Segunda semana del bootcamp: cómo tomar decisiones (`if`/`switch`), repetir trabajo (`for`/
`while`) y estructurar lógica reutilizable con funciones — la base sobre la que se construye
todo el código que viene después.

## 🎯 Objetivos de aprendizaje

Al finalizar esta semana, el estudiante será capaz de:

- [ ] Usar `if`/`else` y el operador ternario con criterio
- [ ] Escribir un `switch` statement clásico y un `switch` expression exhaustivo (Dart 3)
- [ ] Recorrer y repetir con `for`, `for-in`, `while` y `do-while`, controlando el flujo con
      `break`/`continue`
- [ ] Declarar funciones con parámetros posicionales, opcionales (`[]`) y nombrados (`{}`)
- [ ] Escribir arrow functions (`=>`) y reconocer un closure
