# Ebooks Gratuitos — Semana 02: Control de Flujo y Funciones

- [dart.dev — Branches (if, switch)](https://dart.dev/language/branches) — referencia oficial
  de `if`/`switch`, statement y expression
- [dart.dev — Loops](https://dart.dev/language/loops) — `for`, `for-in`, `while`, `do-while`
- [dart.dev — Functions](https://dart.dev/language/functions) — parámetros posicionales,
  opcionales, nombrados, arrow functions y closures
- [Effective Dart](https://dart.dev/effective-dart) — guía de estilo oficial, incluye
  convenciones de nombres y firmas de función
