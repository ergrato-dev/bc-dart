# Videografía — Semana 02: Control de Flujo y Funciones

- [Dart programming language — canal oficial](https://www.youtube.com/@dart_dev) — charlas del
  equipo de Dart, incluye contenido sobre patterns y switch expressions (Dart 3)
- [DartPad](https://dartpad.dev/) — no es un video, pero es el sandbox oficial para probar
  funciones, closures y switch expressions sin instalar nada

> 📌 Prioriza fuentes oficiales (canal `dart_dev`) sobre tutoriales de terceros — el `switch`
> expression y los patterns son de Dart 3 (2023 en adelante), y contenido más antiguo puede no
> cubrirlos o enseñar solo el `switch` statement clásico.
